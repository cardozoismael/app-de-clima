# Proyecto Clima

## Descripción
Aplicación sencilla que muestra el clima actual de una ciudad, usando **HTML, CSS y JavaScript** en el frontend y un **backend de javascript** para consumir la API de OpenWeatherMap.

## Tecnologías
- HTML5, CSS3, JavaScript  

## ⚙️ Funcionalidades
- Búsqueda de clima por ciudad  
- Muestra temperatura, humedad y estado del cielo  
- Interfaz simple y responsive
